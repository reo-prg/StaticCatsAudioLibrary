詳解

ノードクラス

サウンドが出力されるまでの中継地点として使用でき、

通過するサウンドに音量変更やエフェクトをかけることができる

使用方法

ゲームの初期化時に使用するノードを全て生成して、

サウンドが通るツリーを構成しましょう

その際、最終出力に近いノードから構築してください
